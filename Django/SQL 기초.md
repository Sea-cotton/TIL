> # SQL 기초

### SQL commands

- SQL commads 종류
  
  - DDL - 데이터 정의 언어
    
    - CREATE
    
    - DROP
    
    - ALTER
  
  - DML - 데이터 조작 언어
    
    - INSERT
    
    - SELECT
    
    - UPDATE
    
    - DELETE
  
  - DCL - 데이터 제어 언어
    
    - GRANT
    
    - REVOKE
    
    - COMMIT
    
    - ROLLBACK

- SQL Syntax
  
  - 모든 SQL문은 SELECT, INSERT, UPDATE 등과 같은 키워드로 시작하고, **세미콜론(;)** 으로 끝난다.
    
    - 세미콜론은 각 SQL문을 구분하는 표준 방법
  
  - 대소문자 구분x, but 대문자 작성을 권장함.
